# davinC
The best drawn programming language there is. <br>
The goal for this is to make a programming language in which you can submit a png as your code and you will get a png back. <br>
Each color is either a variable, a function, a type, a 'program flow', or nothing. <br>
Goals for this project include: <br>
[] Symbols are cool<br>
[] Write a compiler that works<br>
[] Hello World<br>
[] Fibonacci sequence<br>
[] Coherent <br>
[] Custom function declaration with infinitley expandability<br>
[] Be able to draw a picture around a program and leave it unnafected <br>
[] Have a print statement that actually works <br>
[] Write the compiler in DavinC <br>

My current progress: <br>
I have currently implemented a majority of the code for parsing the png and extracting the meaningful glyphs and leaving the trash behind. I haven't started on the hardest parts yet (tokenization, abstract syntax tree, code generation). My end goal is to make a youtube video about this project to explain how compilers work. My super end goal that will probably be torturously long and horrible is to write the compiler in DavinC.
