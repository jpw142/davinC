# davinC
The best drawn programming language there is. 
The goal for this is to make a programming language in which you can submit a png as your code and you will get a png back. 
Each color is either a variable, a function, a type, a 'program flow', or nothing. 
Goals for this project include: 
[] Symbols are cool
[] Coherent 
[] Custom function declaration with infinitley expandability
[] Be able to draw a picture around a program and leave it unnafected 
[] Have a print statement that actually works 
[] Write the compiler in DavinC
